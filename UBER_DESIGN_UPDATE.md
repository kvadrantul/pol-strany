# Обновление дизайна в стиле Uber Base Design System

## Что было сделано

### 1. Новый интерфейс в стиле Uber
- Полностью переработан UI в стиле Uber Base Design System
- Темная тема с акцентным цветом #09B3AF (Uber Blue)
- Современная типографика и spacing

### 2. Убран режим бригадира
- Удален весь функционал для бригадиров
- Приложение работает только как клиентский интерфейс
- Пользователи автоматически создаются с ролью "client"

### 3. Карта с анимацией
- Статическое изображение карты города в темных тонах
- При нажатии "Найти бригаду" запускается анимация:
  - Одновременно рисуются линии от центра карты к 5 ближайшим бригадам
  - Анимация длится 2 секунды
  - После анимации линии замирают
  - Появляются маркеры на карте

### 4. Список результатов поиска
- После завершения анимации снизу открывается панель с результатами
- Показываются все найденные бригады (до 10)
- Для каждой бригады отображается:
  - Имя
  - Рейтинг
  - Опыт работы
  - Количество выполненных заказов
  - Расчетная стоимость работы
- По клику на карточку открывается Telegram диалог с бригадой

### 5. Скрипт добавления бригад
- Создан Go-скрипт для добавления 20 бригад в базу данных
- Структура:
  - **Простой класс (econom)** - 5 бригад
  - **Комфорт класс** - 5 бригад
  - **Бизнес класс** - 4 бригады
  - **Премиум класс** - 3 бригады
  - **Универсал** - 2 бригады
  - **Самовыравниватель** - 1 бригада

## Структура файлов

### Измененные файлы:
- `index.html` - новый интерфейс с картой
- `styles.css` - стили в стиле Uber Base Design System
- `app.js` - переработанная логика для нового интерфейса

### Новые файлы:
- `scripts/add-contractors.go` - скрипт для добавления бригад
- `scripts/README.md` - инструкция по использованию скрипта

## Запуск

1. Добавьте бригад в базу данных:
   ```bash
   cd scripts
   export DATABASE_URL="libsql://your-database-url"
   go run add-contractors.go
   ```

2. Приложение готово к использованию!

## Особенности

- Анимация на карте работает одновременно для всех 5 бригад
- Результаты поиска могут быть больше 5 (показываются все найденные)
- Интерфейс оптимизирован для мобильных устройств
- Полная поддержка Telegram Web App
