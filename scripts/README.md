# Скрипт добавления бригад

Скрипт для добавления 20 бригад разных классов в базу данных.

## Структура бригад

- **Простой класс (econom)** - 5 бригад
- **Комфорт класс** - 5 бригад
- **Бизнес класс** - 4 бригады
- **Премиум класс** - 3 бригады
- **Универсал** - 2 бригады
- **Самовыравниватель** - 1 бригада

**Итого: 20 бригад**

## Запуск

### Требования

1. Go 1.19 или выше
2. Переменная окружения `DATABASE_URL` должна быть установлена

### Установка зависимостей

```bash
cd scripts
go mod init add-contractors
go get github.com/tursodatabase/libsql-client-go/libsql
```

### Запуск

```bash
export DATABASE_URL="libsql://your-database-url"
go run add-contractors.go
```

Или скомпилировать и запустить:

```bash
go build -o add-contractors add-contractors.go
./add-contractors
```

## Что делает скрипт

1. Подключается к базе данных по `DATABASE_URL`
2. Для каждой бригады:
   - Проверяет, существует ли пользователь с таким `telegram_id`
   - Если нет - создает пользователя с ролью `contractor`
   - Проверяет, существует ли профиль бригадира
   - Если нет - создает профиль с рейтингом, опытом, категориями
   - Если есть - обновляет профиль

## Структура данных

Каждая бригада имеет:
- Имя
- Телефон
- Опыт работы (лет)
- Рейтинг (0-5)
- Количество выполненных заказов
- Категорию (класс тарифа)
- Telegram ID

## Примечания

- Скрипт безопасен для повторного запуска - он обновит существующие записи
- Telegram ID уникальны для каждой бригады
- Все бригады создаются как активные (`is_active = true`)
