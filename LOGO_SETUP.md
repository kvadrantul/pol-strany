# Настройка логотипа

## Текущий статус

Я создал SVG версию логотипа на основе описания: `public/images/logo.svg`

## Если у вас есть оригинальный файл логотипа:

1. Сохраните файл логотипа в папку `public/images/`
2. Назовите файл `logo.png`, `logo.jpg` или `logo.svg`
3. Обновите путь в `public/index.html` если нужно:

```html
<img src="images/logo.png" alt="Пол Страны" class="logo-image">
```

## Поддерживаемые форматы:

- ✅ SVG (рекомендуется - масштабируется без потери качества)
- ✅ PNG (с прозрачностью)
- ✅ JPG

## Размеры:

Рекомендуемый размер: 200x200px или больше (будет автоматически уменьшен до 120x120px в интерфейсе)

## Текущая реализация:

Логотип отображается на экране выбора роли с эффектом тени для лучшей видимости на градиентном фоне.
